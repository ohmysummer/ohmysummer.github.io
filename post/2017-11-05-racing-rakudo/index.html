snapper 函数以数据采集的时间为间隔。程序终止后，会显示出上面的表格。

该模块自带大量的 sub，可以收集手头相同的数据，并提交自己的报告。在长期运行的过程中可能是合理的。或者你时不时的手动调用 report sub。

use Telemetry;

react {
    snapper;
    whenever Supply.interval(60) {
        say report;
    }
}

如果终端不能切入，你可以使用 http 来获取遥测数据。
