使用 Perl 6 轻松活泼地写饭否机器人 安装 Perl 6 Perl 6 是一门很有趣的语言。 目前支持 Windows, macOS, Linux 主流操作系统, 首先去官网下载对应的安装包, 然后安装。安装完成后在终端/命令行提示符中输入 perl6 -v 它会打印出 Perl 6 的版本信息： This is Rakudo Star version 2018.04.1 built on MoarVM version 2018.04.1 implementing Perl 6.c.
